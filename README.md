# MachineLearningAmateur.github.io

Root GitHub Pages site for `https://machinelearningamateur.github.io/`.

## Local structure

- `index.html` is the homepage.
- `styles.css` holds the site styling.
- `.nojekyll` tells GitHub Pages to serve the repo as-is.

## Publish

1. Create a GitHub repository named `MachineLearningAmateur.github.io`.
2. Add that repo as `origin`.
3. Push the `main` branch.

Example:

```powershell
git remote add origin https://github.com/MachineLearningAmateur/MachineLearningAmateur.github.io.git
git add .
git commit -m "Initial site"
git push -u origin main
```
